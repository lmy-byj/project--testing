# GitHub 上传步骤（等你注册好后按这个做）

## 第一次上传代码

1. 登录 github.com
2. 右上角 + → New repository
3. Repository name: `amxm-life`
4. 选 Private（私有，只有你能看）
5. 点 Create repository

6. 进入仓库页面，点 "uploading an existing file"
7. 把整个 `阿铭小梦生活指南` 文件夹里的内容拖进去
8. 点 Commit changes

## 连接 Netlify

1. 登录 netlify.com
2. Add new site → Import an existing project
3. Connect to GitHub → 选 `amxm-life` 仓库
4. Build settings 全部默认（不用改）
5. Deploy site

完成后会得到一个网址，这就是你们两个人共用的 App 地址。

## 以后更新代码

我改好代码后会给你新的文件，你在 GitHub 网站上把对应文件替换掉，
Netlify 会自动在2分钟内更新网站。
